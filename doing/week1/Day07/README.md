3. 计算 n 条直线最多能把平面分成多少部分？（n >= 1）

4. 猴子第一天摘了若干个桃子，当即吃了一半，还不解馋，又多吃了一个；第二天，吃剩下的桃子的一半，还不过瘾，又多吃了一个；以后每天都吃前一天剩下的一半多一个，到第10天想再吃时，只剩下一个桃子了。  
问第 i（i 的取值范围为 [1, 10]）天的桃子个数？
